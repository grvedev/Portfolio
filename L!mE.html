Core Capabilities

Built for developers
who ship fast.

01

Cold-Start in Milliseconds

Containerized emulation cores launch in under 3ms. No pre-warming, no waiting — your users get instant access every single time.

02

Global Edge Network

Sessions run from 40+ PoPs worldwide. Inputs route to the nearest node, slashing latency for a native-feel experience anywhere on earth.

03

REST + WebSocket API

A clean, versioned API to launch, control, snapshot, and stream emulation sessions. SDKs for Node, Python, Go, and Rust included.

04

Persistent Save States

Snapshot any session to persistent storage. Resume exactly where you left off — across devices, across users, across months.

05

Isolated Sandboxes

Every session runs in a gVisor-hardened container. Full memory isolation, network controls, and audit logging baked in from day one.

06

Live Session Streaming

Stream emulation output as a low-latency WebRTC or HLS feed. Embed playback in your app with our zero-dependency viewer component.

Integration

Three lines of code
to full emulation.

01

Authenticate

Grab your API key from the dashboard. We support API keys, JWTs, and OAuth 2.0 for enterprise customers.

02

Launch a Session

POST to /v1/sessions with your target platform and ROM reference. A session URL returns in ~2ms.

03

Stream & Control

Open a WebSocket to stream video output and send controller input in real-time. Save states anytime.

04

Scale Without Limits

Sessions auto-scale to thousands of concurrent users. Usage-based pricing means you only pay for what runs.
